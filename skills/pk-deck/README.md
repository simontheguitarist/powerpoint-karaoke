# pk-deck

A Claude Code skill that generates a PowerPoint Karaoke deck — a folder of intentionally absurd HTML slides on any topic, zipped and ready to upload.

## Install

Extract this folder into `~/.claude/skills/` so the path becomes `~/.claude/skills/pk-deck/SKILL.md`.

## Use

In any Claude Code session, run:

```
/pk-deck
```

…then tell it your topic, slide count (5–20), and spice level (mild / medium / spicy). Or trigger by typing what you want directly:

> Make a pk deck about the secret life of paperclips, 12 slides, spicy.

Claude will write a folder of HTML slides, zip it, and tell you the path. Upload the zip in your PowerPoint Karaoke app at **Studio → Upload**.
